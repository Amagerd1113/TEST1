## 📊 性能数据建议

### 当前 README 声称
```
Success Rate: 77.4% (可能过于乐观)
SPL: 71%
推理时间: <5ms
参数量: <500K
```

### 建议修正为（保守估计）
```
Success Rate: 52-58% (需验证)
SPL: 24-28%
推理时间: ~20ms
参数量: ~85-95M (freeze backbone)
```

**与 SOTA 对比** (2024-2025):
- RATE-Nav (CVPR 2025): 67.8% SR, 31.3% SPL
- NavFoM (2025): 45.2% SR (zero-shot)
- OpenVLA (2024): ~70% SR on manipulation
